📘 How to Connect

🔹 Method 1: QR Code

  1. Go to WhatsApp → Settings → Linked Devices.
  2. Tap Link a Device.
  3. On this panel, click Generate QR Code.
  4. Scan the QR code with your phone.

🔹 Method 2: Pair Code

  1. In WhatsApp, go to Linked Devices → Link with phone number.
  2. Select your country and enter your WhatsApp number.
  3. Click Get Pair Code on this panel.
  4. Copy the pair code and paste it into WhatsApp.
